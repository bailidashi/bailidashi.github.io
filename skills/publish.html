publish

自动提交并上传网站到 GitHub —— 一行命令替代 GitHub Desktop

触发命令 /publish 类型 自动化部署 路径 D:\mygit\bailidashi.github.io 📥 下载 SKILL.md

执行流程

1git status
→
2git add -A
→
3git commit
→
4git push
→
✓已发布!

行为细节

  • cd 到项目根目录
  • 查看变更文件，根据改动自动生成中文 commit 信息
  • 推送前确认用户意图，避免误提交敏感信息
  • 推送到 main 分支，约 1-2 分钟后 GitHub Pages 生效
  • 若 push 失败（网络问题），告知用户具体错误并建议重试

💡 使用场景：每次改完网站代码后，输入 /publish 就能自动走完 git add → commit → push 全流程，不用再打开 GitHub Desktop 了。
